JavaScript If and If-Else Statements

In JavaScript, the `if` statement is used to make decisions based on conditions. If a certain condition is true, the block of code inside the `if` statement will be executed. Optionally, an `else` block can be added to execute code when the condition is false.

Example 1: Simple If Statement

            
                // Simple if statement
                var num = 10;
                if (num > 0) {
                    console.log("The number is positive.");
                }

Example 2: If-Else Statement

            
                // If-else statement
                var hour = new Date().getHours();
                if (hour < 12) {
                    console.log("Good morning!");
                } else {
                    console.log("Good afternoon!");
                }

Example 3: If-Else If-Else Statement

            
                // If-else if-else statement
                var time = new Date().getHours();
                if (time < 12) {
                    console.log("Good morning!");
                } else if (time < 18) {
                    console.log("Good afternoon!");
                } else {
                    console.log("Good evening!");
                }

These are basic examples of using `if` and `if-else` statements in JavaScript. You can customize and expand upon these examples based on your specific needs.
