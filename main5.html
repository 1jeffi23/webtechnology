JavaScript String Properties

In JavaScript, strings are a sequence of characters, and they have various properties and methods that you can use to manipulate and retrieve information about strings.

String Length Property


    // String Length Property
    var greeting = "Hello, World!";
    console.log(greeting.length); // Output: 13

String Concatenation


    // String Concatenation
    var firstName = "John";
    var lastName = "Doe";
    var fullName = firstName + " " + lastName;
    console.log(fullName); // Output: John Doe

String Indexing


    // String Indexing
    var message = "JavaScript";
    console.log(message[0]); // Output: J
    console.log(message.charAt(4)); // Output: S

String Conversion


    // String Conversion
    var number = 42;
    var strNumber = String(number);
    console.log(strNumber); // Output: "42"

These are just a few examples of the properties and operations you can perform on JavaScript strings. Explore more string methods and properties based on your needs!


JavaScript String Methods

In JavaScript, strings have various methods that you can use to manipulate and retrieve information about strings.

Method Description
slice(start, end) Extracts a part of a string and returns a new string.
substring(start, end) Extracts a part of a string between two specified indices.
substr(start, length) Extracts a specified number of characters from a string, starting at a specified position.
replace() Replaces a specified value with another value in a string.
toUpperCase() Converts a string to uppercase letters.
toLowerCase() Converts a string to lowercase letters.
concat() Concatenates two or more strings.
trim() Removes whitespace from both ends of a string.
trimStart() Removes whitespace from the beginning of a string.
trimEnd() Removes whitespace from the end of a string.

Explore these string methods based on your needs!
