JavaScript Objects

In JavaScript, an object is a collection of key-value pairs. It is a versatile data type that can store various types of data.

Example 1: Creating an Object


    // Creating an object
    var person = {
        firstName: "John",
        lastName: "Doe",
        age: 30,
        isStudent: false
    };

Example 2: Accessing Object Properties


    // Accessing object properties
    console.log(person.firstName); // Output: John
    console.log(person["lastName"]); // Output: Doe

Example 3: Modifying Object Properties


    // Modifying object properties
    person.age = 31;
    person["isStudent"] = true;

Example 4: Adding New Properties


    // Adding new properties to an object
    person.city = "New York";
    person["email"] = "john@example.com";

Example 5: Object Methods


    // Object methods
    person.fullName = function() {
        return this.firstName + " " + this.lastName;
    };

    console.log(person.fullName()); // Output: John Doe

This is a basic introduction to JavaScript objects. Objects in JavaScript are powerful and can be used to represent complex data structures and functionalities.
