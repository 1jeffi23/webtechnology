JavaScript Arrays

In JavaScript, an array is a special variable that can hold multiple values. Each value is called an element, and you can access them using an index.

Example 1: Creating an Array

            
                // Creating an array
                var fruits = ["Apple", "Banana", "Orange", "Mango"];

Example 2: Accessing Array Elements

            
                // Accessing array elements
                console.log(fruits[0]); // Output: Apple
                console.log(fruits[2]); // Output: Orange

Example 3: Modifying Array Elements

            
                // Modifying array elements
                fruits[1] = "Grapes";
                console.log(fruits); // Output: ["Apple", "Grapes", "Orange", "Mango"]

Example 4: Array Methods

            
                // Array methods
                fruits.push("Pineapple");
                fruits.pop();
                fruits.unshift("Watermelon");
                fruits.shift();

This is a basic introduction to JavaScript arrays. Arrays in JavaScript are versatile and can be used to store and manipulate collections of data.
